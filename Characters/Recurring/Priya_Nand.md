## PERSONALITY

- Literal-minded; irritated when displays use friendly labels for unfriendly returns

## PATTERN RECOGNITION HABITS

- Stacks partial returns mentally before vocalizing
- Notices **absence** (dropout, collapse, missing harmonics) before **presence**
- Re-runs the same sweep when uneasy—not superstition, procedural stubbornness
- Keeps a handwritten strip log when she distrusts the auto-summary (Holt allows it)

---

## STRESS BEHAVIOR

- Speech compresses: clipped, overly literal, fewer qualifiers
- Stops saying *might*; starts saying *did* or *didn't*
- Hands stay on controls; body still
- Will repeat the same readout until someone acknowledges the specific gap

---

## COMMUNICATION STYLE

- Brief, clipped, literal
- *"That's not what the return says."*
- *"We lost the line."* not *"Something happened."*
- Does not small-talk on watch; does not silence comfortably

---

## HOW SHE INTERPRETS INCOMPLETE DATA

- Reports **what the array did not show** as loudly as what it did
- Treats single-channel confirmation as non-confirmation
- Prefers *unknown contact* over invented class labels
- Accepts emotional shock late—after the pattern is already broken
