# PROSE STYLE SPECIFICATION — BOOK ONE

## CORE VOICE PRINCIPLE
The narrative voice is controlled, precise, and observational. Emotion is present but not indulgent.

## STYLE DNA MIX
- Asimov: clarity, logic-first exposition
- Clarke: conceptual awe, restrained emotional language
- Clancy: operational realism and procedural detail
- Tolkien/Jordan: structural gravity and mythic undertone (subtle)

## SENTENCE STRUCTURE RULES
- Medium-length declarative sentences
- Short sentences for tension spikes
- Longer sentences only for technical explanation

## PROHIBITED
- sarcasm, slang, casual tone
- excessive metaphor unrelated to systems

## DESCRIPTION RULES
- ships = functional
- space = scale + silence
- anomalies = effects first, cause unknown
