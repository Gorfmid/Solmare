# THE 70/30 RULE — CREW / ANOMALY RATIO

**Also called:** crew-first ratio · anomaly-as-weather rule

**Companion doc:** `global_writing_principles.md` — applies to all chapters; this rule adds the crew/anomaly ratio from Act II onward.

Applies to **Book One from Chapter 6 forward** (Act II and beyond). Chapters 1–5 were anomaly-establishment; ratio inverts afterward.

---

## Rule

Every chapter should **feel** roughly like:

| ~85% | ~15% |
|------|------|
| Crew | Direct anomaly focus |
| Mission | |
| Relationships | |
| Problem-solving | |
| Politics | |
| Exploration | |
| Action | |
| Professional life | |

The rule is named **70/30** for **intent**: human operational story dominates; anomaly is pressure, not protagonist. The operational target on the page is **~85/15**, not literal word-count math.

---

## Anomaly as Weather

> Nobody is surprised that it rains. They're interested in how the crew operates in the rain.

**Bad escalation:** new anomaly every chapter; surprise stacking; mystery as main character.

**Good escalation:** same rules, harder consequences; crew adaptation; competence under inconsistent observation.

The anomaly **causes** problems. It should not **occupy** every page.

---

## Human Victory Requirement (Ch. 6+)

Each chapter should end with a **human victory**—competent people solving a difficult problem—not merely an anomaly revelation.

Victory ≠ solving the cosmology. Victory = crew earns something through collaboration:
- reconstructed data
- trustworthy path confirmed
- person or asset recovered
- conspiracy exposed (fact, not full motive)
- successful evasion or standoff

Pair victory with one unanswered question. Mystery stays alive because **people** matter, not because confusion increases.

---

## Engine Check

Before marking a chapter complete, ask:

1. Would a reader remember what the **crew did** more than what the anomaly did?
2. Is there at least one moment of professional problem-solving with a clear outcome?
3. Does the anomaly appear as **environment** (weather) rather than **event** (plot twist parade)?

If anomaly content exceeds ~15% of reader attention, cut exposition and add crew operational life.
