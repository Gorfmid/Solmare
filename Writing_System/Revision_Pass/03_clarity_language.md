# Revision Pass 3 — Clarity / Language

**Router:** `Writing_System/revision_pass.md`  
**Load only this file during Pass 3.** Passes 1–2 should already be done.

---

## Purpose

Plain speech on first hearing. Cut captions, soft-noun meta, staff poetry, cute synonym rotation. Expand the real fight.

**Author mid-read rule:** If the author asks *what does this mean?* or *this is weird*, treat that line as a hard fail — rewrite in plain speech immediately, then check neighbors. Do not defend the shorthand.

**Note:** The full first-hearing **preflight checklist** runs in Pass 4. During Pass 3, plain-expand every foggy line you hit.

---

### 3.1 Cut clever captions after the work is already shown

If action, seating, procedure, or dialogue already carries the beat, cut the narrator wink that explains it.

**Also later:** Pass 6 (`06_ai_signature.md`) strips the same habit as **AI Signature Removal** after Character Pass — meaning sentences after strong images, over-articulate speech, excess narrative certainty. Apply §3.1 whenever you hit it; Pass 6 is the dedicated full strip.

| Weak | Prefer |
|------|--------|
| People-reading disguised as furniture. | Cut. Keep the chair move. |
| People-judgment as furniture again. Useful furniture. | Cut. Keep the seating note. |
| Soft wisdom after two facts | End on the fact or the next action |

**Test:** If removing the last sentence leaves the scene clearer, remove it.

**Also — do not stack room atmosphere.** One useful concrete detail beats a paragraph of gray walls + bolted tables + bright screws + three smells. If the previous paragraph already said the tables were bolted, do not say it again. Cut resin/crate/anteroom tourism unless a character uses it.

| Weak | Prefer |
|------|--------|
| dull institutional gray of a corridor station built for packet handoff and emergency docking, not for politics + bolted tables + second display + recycled air and machine oil and faint resin of Dominion packing seals | institutional gray; screws still bright on a new display; recycled air and machine oil |

**Test:** Can the reader see the room in one breath? If the paragraph needs a second breath for smell alone, cut.

### 3.1a Cut self-denying analogies

Do not explain a behavior by comparing it to the thing you then claim nobody is admitting.

| Weak | Prefer |
|------|--------|
| Orlen watched him the way a professional watches a subject without admitting the watch | Orlen watched how he handled the unfamiliar seasoning, then wrote one line when he looked away |
| X did Y the way Z does W without admitting it | Show the concrete beat: what they looked at, what they wrote, what they delayed |
| It was not quite accusation / not quite permission / not quite a smile (as caption) | Name the tone once, or show the next action — do not half-take the label back |

**Pattern to kill:** *does [behavior] the way a [role] does [same behavior] without admitting [behavior].*

### 3.1b Ban *soft nouns* / *label* / invent-words meta talk

Do **not** write stylish shorthand about wording, classification, or readiness:

- *soft nouns*, *soft language*, *soft term*, *soft synonym*, *clean noun*, *wrong nouns*
- *invent words*, *invent a phrase*, *tomorrow's words*
- *finished label*, *better label*, *incomplete label*, *working label*, *fix the label*, *earned one* (as classification poetry)
- *the noun*, *the verb*
- *strategic gift*, *as a gift*, *not a gift* (disclosure metaphor)

| Weak | Prefer |
|------|--------|
| finished label / better label / incomplete label | finished answer / better category / incomplete classification — or name the real thing (*distress*, *hold weapons*) |
| when the table has earned one | when we have one / when both sides agree |
| invent words for next week | do not fake next week's readiness |
| soft nouns under pressure | guess under pressure / a word that means two actions |
| not a full strategic gift / not as a gift | not a full strategic brief / under limits / not free access |
| *fringe* as default border noun | *outer corridor* / *border* / *Neutral Zone edge* — keep *fringe* only if a locked map name requires it |

**Keep *label* only** for literal objects (mug label, pedestal labeled for Rowan) or rare technical stamps — never as atmospheric talk about meaning.

**Keep *gift* only** for literal objects (Lisette's mug). Never for maps, archive access, or disclosure.

**Test:** If the sentence is about *labels* or *gifts* instead of the concrete failure or ask, rewrite. Never reintroduce this cluster after cutting it.

### 3.2 Dumb down institutional jargon — then expand the real fight

Do not leave the reader to decode bureaucracy labels.

| Weak | Prefer |
|------|--------|
| Recording rights still unresolved. | Name both sides’ ask: full audio vs written summary; dual consent for raw sound. |
| veto language / stop rules / who can walk (as cute shorthand) | who can end a session / who can leave the table |
| shared nouns / shared terms (as cute shorthand) | word lists / what both sides are allowed to say / operational words |
| stop rules and shared nouns / walk-outs and word lists | who can leave and what words mean — name the work, not the label pair |
| provisional instruments (when first introduced) | Say what they do in one plain sentence, then keep the term if useful |
| *header* as default for any traffic / classification / status message | *status line*, *message*, *classification line*, *shared traffic*, *stamp* — name what the reader needs to see |
| soft term gets into a header | soft term gets into a status line |

**Rule:** First say the **dispute in plain speech**. Labels may follow.

**Also — *header* is staff jargon.** Officers may think in headers; readers should not have to. In dialogue and narration, default to the plain object (*status line*, *message*, *classification*). Keep *header* only when the beat is literally a field name on a form or protocol stack and a specialist is naming that field — and even then, prefer plain first.

**Also:** Do not stack neat institutional shorthand in POV or narration (*stop rules and shared nouns*, *who can walk*, *custody and headers*). If the character hates the assignment, show the hated **work**, not a tidy pair of staff terms.

| Weak (tidy staff talk) | Prefer (hated work) |
|------------------------|---------------------|
| an envoy with stop rules and shared nouns | an envoy arguing over who can leave and what words mean |
| Field judgment. Who can walk. | Field judgment. Who can end a session. What words both sides may use. |

**Plain speak over staff poetry:** Keep the idea. Cut the elevated phrasing. If a sentence needs unpacking to understand what the character believes, rewrite it shorter and plainer.

| Weak (overwritten) | Prefer (plain) |
|--------------------|----------------|
| the right to refuse a clean story when the board would not settle | not claiming more than the instruments showed |
| refusing to invent certainty the instruments had not earned | not claiming more than the instruments showed |
| provisional categories as moral architecture | watching longer than politics preferred |
| bought X a small pool of speakers who can… | was long enough for X to train a few speakers. Those people can… |
| keep the nouns from killing people | make sure a bad word does not start a fight |
| temporary permission wearing permanent bolts | bolts permanent; permission for politics temporary |
| felt like penance / buy penance with an adjective | felt like making up for the mistake |
| silence that functioned as permission | silence that meant they could keep planning |
| mission-critical / not decorative | can kill people if wrong / not optional |

**Test:** Would this officer say it in a ready room, or only in a style guide? Ready-room speech wins. Can the reader get the fact on first pass? If not, break the sentence.

**Do not hammer a plain phrase:** Once you have said *who can end a session* (or any cleared plain term) clearly once, later beats can shorten (*suspension*, *exit rules*, *seating*) instead of repeating the full phrase every paragraph.

### 3.2a *Argue* / *arguing* habit

Default: **do not** use *argue* / *arguing* as the easy verb for politics, capitals, lexicon work, or process.

| Weak | Prefer |
|------|--------|
| Helion and Civil Government have finished arguing | have reached a decision / signed off / closed deliberation |
| already arguing about which noun | already at it over which noun / still working the wording |
| took it without arguing | took it without comment / without protest |
| cheapest argument (as process talk) | cheapest claim / easiest line |
| arguing with the coffee urn (anyone but Brenner) | name the maintenance fact |

**Brenner only (sparingly):** Brenner may *argue* with welds, urns, controllers — literal fight with hardware that matters. Do not spread that habit to other mouths. Do not stack *argue* lines in the same scene.

**Allowed elsewhere only** when a real person-to-person dispute is the point and no plainer verb works (*Rowan did not contest the point*). If you can say *disputed*, *pushed back*, *refused*, *worked the wording*, prefer that.

**Search:** `\barguing\b|\bargue\b|\bargued\b` — keep Brenner hardware lines; rewrite institutional / habit uses.

### 3.2b Inanimate objects do not *pretend*

Do **not** give traffic, stations, boards, rooms, weather, or other non-people agency that *pretends*, *knows*, *hopes*, or similarly performs human intention.

| Weak | Prefer |
|------|--------|
| Morning traffic still *pretended* the station was only bolts and packet handoff | Morning traffic still looked like ordinary bolt-and-packet work |
| the quiet *pretended* to be safety | the quiet looked like safety / people treated quiet as safety |
| the board *pretended* nothing was wrong | the status display stayed quiet / showed nothing elevated |
| the room *pretended* civility | people kept civil / the room stayed formal |

**Allowed:** People pretend (*do not pretend the board agrees with the headline* — a human instruction).

**Test:** If the subject cannot intend, it cannot pretend. Rewrite with look / show / stay / people treating it as.

**Search for regressions:** `\bpretended\b|\bpretending\b` (narration) · `traffic .*pretend|station .*pretend|board .*pretend|room .*pretend|quiet .*pretend`

### 3.2c Ban *lying* / *honest* as default metaphor

Do **not** reach for *lie / lied / lying / lies / honest* as easy rhetoric for overcommitment, dual roles, bad readings, incomplete briefs, or institutional mismatch.

| Weak | Prefer |
|------|--------|
| cannot run a yard and a reactor without lying to somebody | cannot run a yard and a reactor. Pick one. |
| knowing which panel lies | knowing which panel looks fine and is not / fails quiet / will not settle |
| the urn was honest / panels kept honest | the urn worked / caught what his boards missed |
| finished lying about size / the return lies | before the mass return settled / the lock will not hold / the profile keeps shifting |

**Keep *lie* only** when a person is actually deceiving someone on the page.

**Instruments and plots do not lie.** They fail to settle, refuse a lock, read clean when they are not, or keep shifting.

**Search:** `\blie\b|\blied\b|\blying\b|\blies\b|\bhonest\b` — keep rare real deception; rewrite metaphor / instrument / capacity uses.

### 3.2d Ban default *the board* — and do not replace it with *wall* spam

Do **not** use *the board* as a vague catch-all for every display, queue, or status surface. Name the actual instrument in that scene.

Do **not** then hammer *traffic wall* / *status wall* as the new default noun every paragraph. One clear name per scene is enough; after that, *the display*, *the plot*, *the queue*, *Segment traffic*, or the action (*Park put the packet up*).

| Weak | Prefer |
|------|--------|
| watched the board stay quiet | watched the status display / relay display stay quiet |
| kept his eyes on the board | kept his eyes on the shared traffic display / packet queue |
| put a packet on the board | put a packet into the queue / on the shared display |
| Hold the board | Keep the traffic moving / Hold the room |
| joint board / Segment board | shared traffic display / Segment status display |
| approach board | approach plot / approach display |
| secondary board (comms) | secondary console / relay console |
| *traffic wall* every other sentence | name it once, then *the display* / *the plot* / *the packet* |

**Keep *board* only** for:
- the verb (*board the ship*)
- a literal named object already established in that scene (*injury board*, *duty roster*) when that exact object is the point
- rare specialist shorthand in dialogue once the concrete instrument is already clear

**Keep *traffic wall* / *status wall* rare** — useful once when the big shared surface is first on stage. Not a synonym to rotate for every glance at a screen.

**Test:** Could a reader who has never read Fleet jargon picture the object? If not, name it. If every sentence says *wall*, cut half of them.

**Search:** `\bthe board\b|\bon the board\b|\boff the board\b|\btraffic wall\b|\bstatus wall\b`

### 3.2e Ban default *petition*

Do **not** use *petition* as a vague institutional noun for every civilian ask. Name the actual request.

| Weak | Prefer |
|------|--------|
| petitioned the same wall for two years | been asking for a proper school building for two years |
| wrote the petition again | wrote the school request again / asked for leak-proof walls again |
| School petition: automated acknowledgment | School building request: automated acknowledgment |
| petition list | repair list / request list — or name the item (*beacon array*) |

**Keep *petition* only** when the legal/process form itself is the point (someone is literally filing a named petition type, or fighting the form).

**Test:** What are they actually asking for? Say that.

**Search:** `\bpetition\b|\bpetitions\b|\bpetitioned\b` — rewrite vague uses to the concrete ask.

### 3.2f Ban default *hold* / *holds* / *held*

Do **not** use *hold* as a vague default verb for endurance, weatherproofing, station-keeping, custody, or “still true.” Name the actual action.

| Weak | Prefer |
|------|--------|
| walls that hold weather | walls that withstand weather |
| If today holds | If today stays quiet / If the talks stay on track |
| Survey window still held | Survey window still open |
| freighters hold their routes | freighters stay on their routes |
| *Graylane* held | *Graylane* stayed on station |
| landmass hold its green | landmass stay green |
| plot held a thin line of shoots | plot showed a thin line of shoots |
| still holds shape | still keeps its shape |
| *holds* (custody) | *has … in custody*, *logged*, *sealed* |

**Keep *hold* only** when it is a locked technical term the scene is fighting (*overnight hold*), a literal grip, or no plainer verb works after a real try.

**Test:** Hold *what*, against *what*? Say that.

**Search:** `\bhold\b|\bholds\b|\bholding\b|\bheld\b` — rewrite vague endurance / custody / station uses.

### 3.2g Ban default *empty* / *went dark* / *returns stopped*

Do **not** use these as the default way to say a colony, channel, or bearing has been lost:

- *empty* (*empty bearing*, *empty plot*, *empty channel*, *went empty*, *Empty is what we have*)
- *went dark* / *gone dark*
- *returns stopped* / *expected returns missing* / *returns missing*

Name the concrete failure.

| Weak | Prefer |
|------|--------|
| On that bearing — empty | No answer on that bearing / Founders' Reach is missing |
| Colony channel empty | Colony channel: no answer |
| Expected returns missing | Settlement beacons gone / no traffic where the colony should be |
| long watch went empty | long watch lost the colony / long watch has no answer |
| the place went dark | the colony was attacked / taken apart — see §3.2j |
| Prepare for the next empty bearing | Prepare for the next colony on the path |

**Keep *empty*** only for literal containers, rooms, hands, cups, or a seat left open on purpose — not as loss shorthand for a missing world.

**Test:** Name the concrete failure. Do **not** default to *went silent* / *stopped answering* once the attack is known — see §3.2j.

**Search:** `\bempty\b|went dark|gone dark|returns (stopped|missing)|Expected returns`

### 3.2h Ban default *shared warning format* / *warning format*

Do **not** use staff shorthand for the joint Fleet–Dominion packet path as if the reader already lives in Session Three.

Avoid as default:

- *shared warning format*
- *warning format* (when it means the joint packet path)
- *shared format* / *joint format* (same fog)

Say where the package went and what “shared” means in plain speech.

| Weak | Prefer |
|------|--------|
| She sent the package under the shared warning format | She sent the package to Contact Node Seven — same words for Fleet and Dominion |
| while we finished the warning format | while we finished the joint traffic words / while both sides could finally read the same packet |
| today's shared format | today's joint channel / the shared display both sides can see |

**Keep** *warning* when a character is literally talking about a warning. Cut *format* as the magic noun that stands in for the whole instrument.

**Test:** Can a reader who skipped the lexicon scenes still know what was sent, to whom, and who can read it? If not, rewrite.

**Search:** `shared warning format|warning format|shared format|joint format`

### 3.2i Ban default *lag* / *lag voice*

Do **not** use *lag* as the default word for remote officers, pedestal channels, Helion patches, or “not in the room.”

Especially kill:

- *lag voice* / *on lag* / *stayed on lag* / *orders on lag* / *came on lag*
- *when lag and sensors disagree* (name Helion’s guidance / the channel / delayed orders)

| Weak | Prefer |
|------|--------|
| Rowan's lag voice arrived thinner than usual | Rowan came through thinner than usual |
| A younger Fleet officer on lag said | A younger Fleet officer on the channel said |
| Rowan stayed on lag | Rowan stayed on the channel / stayed patched in |
| contradictory orders on lag | contradictory orders from Helion / over the channel |
| when lag and sensors disagree | when Helion's guidance and our sensors disagree |

**Keep** *lag* / *delay* only when **time delay itself** is the beat (*the delay was long enough that interruption felt like rudeness*). Ordinary hesitation is *pause* / *moment*. For *hold/holds/held*, see §3.2f.

**Test:** If *lag* only means “remote person speaking,” rewrite. If the sentence is about delay as the operational problem, *delay* is allowed; *lag voice* is not.

**Search:** `\blag\b|lag voice|on lag|stayed on lag`

### 3.2j Ban default *went silent* / *stopped answering* for a known attack

Do **not** keep saying a colony *went silent* or *stopped answering* once the crew (and the reader) know it was attacked — especially Founders' Reach after live observation.

Especially kill as habit:

- *the colony went silent* / *when the colony went silent*
- *a colony stopped answering* / *until it stops answering* (as soft substitute for *attacked*)
- rotating *silent* / *no answer* / *stopped answering* for the same spent fact

| Weak | Prefer |
|------|--------|
| We were not ready when the colony went silent | We were not ready when the colony was attacked |
| That a colony stopped answering | That a colony was attacked |
| places people ignore until it stops answering | places people ignore until the hulls arrive / until it is taken apart |
| the colony went silent; then we watched it leave | the colony was attacked; then we watched it leave |

**Keep** *no answer* / *stopped answering* only for the early unknown — before anyone has watched the attack, or for a channel that literally will not reply and attack is not yet the point.

**Test:** If they saw the Peligar take it apart, say *attacked* / *taken apart* / *watched it leave*. Do not hide behind silence vocabulary.

**Search:** `went silent|gone silent|stopped answering|stops answering`

### 3.2k Ban *do not invent what you have not seen* sermons

Do **not** write preachy observation morals that sound like staff poetry:

- *do not invent what you have not seen*
- *do not invent what you cannot certify*
- *do not invent certainty* (as dialogue sermon — already banned as style meta in §3.1b)
- *contradictory orders* / *when orders and sensors disagree* as default Helion fog

Characters may refuse Helion’s hunger for blame. They do it in plain work speech — see §3.2l for *war story*.

| Weak | Prefer |
|------|--------|
| Hold your watch. Do not invent what you have not seen. | Stay on watch. / Confirm before you name it. |
| Expect contradictory orders within the day | They will want ships, answers, and someone to blame |
| Local judgment still stands when their orders and our sensors disagree | If Helion wants us fighting a fight we have not confirmed, we stay with the plot |

**Test:** If the line is a moral about inventing / certifying / earning observation, cut it to the real order.

**Search:** `invent what you|do not invent|contradictory orders|orders and (our )?sensors disagree`

### 3.2l Ban default *war story*

Do **not** use *war story* as the default catchphrase for Helion / Council / gallery hunger.

Especially kill as habit:

- *Helion wanted a war story*
- *they will want … a war story*
- *ask you for a war story*
- *cut war words* / *private war words* (when it only means invasion talk)

Name the real ask: *someone to blame*, *invasion language*, *fighting before we confirm*, *an enemy with a face*.

| Weak | Prefer |
|------|--------|
| Helion wanted a war story | Helion wanted someone to blame |
| They will want ships, answers, and a war story | They will want ships, answers, and someone to blame |
| They will ask you for a war story before you know what is happening on the next one | Helion will want you fighting before you know what this new alert is |
| Thessaly cut war words before they stuck | Thessaly kept invasion talk out of the first lines |

**Keep** *war* only when the character is literally talking about war, fighting, or a declared conflict — not as a cute noun for bad institutional narrative.

**Test:** If *war story* can be replaced by *someone to blame* / *invasion talk* / *fighting before we confirm*, do that. Do not stack the phrase three times in one chapter.

**Search:** `war story|war words`

### 3.3b Cut soft *wrong* talk and bin metaphors in narration

Do not use *wrong* as atmosphere, mood paint, or a vague emotional noun. Do not say a character “had no bin” unless they are literally talking classification work aloud.

| Weak | Prefer |
|------|--------|
| Park felt the same wrong recognition and had no bin for it | Park saw it too. Same hands. Same eyes. |
| incomplete in a way that still sat wrong with him | provisional and unknown. He still did not have a better category. |
| still wrong against every human emergency format | still matched no human emergency format |
| the stun held / the wrongness landed | name the silence, the stare, or the next action |
| quieter wrongness | cut; say the fact (company / already running / etc.) |
| wrong enough to note | irregular enough to note / off-spec enough to log |
| shifting wrong for weather | shifting in ways weather did not explain |
| the same wrong intensity | the same intensity / the same unresolved intensity |

**Test:** If *wrong* can be deleted and the sentence still means the same concrete thing, delete it. If the beat is classification anxiety, say *category*, *label*, or *format* — once — then move on.

Also already in Writing Pass 1 (apply early if they confuse the scene): gloves-as-tell · meta noun/verb talk · instrument “lies” / *lying* metaphor (now also Pass 3 §3.2c).

### 3.3f Plain speech, less hyperbole (Ch.3 live-edit distillate)

Distilled from Book Two Ch.3 author line edits (2026-07-20). Apply on every Revision Pass and when the author is mid-read fixing foggy lines.

**Voice target:** Plainer. Less hyperbole. Less costume metaphor. Blunt where the fight is real. Expand when a clever compression hides the meaning.

| Weak (clever / compressed) | Prefer (plain / expanded) |
|----------------------------|---------------------------|
| *salute our archives* | not asking Fleet to accept our records as finished fact |
| *edge does not hold still for your survey edition* | a line that moves before your charts can finish printing it |
| *enemy with a face* / *fails under longer watch* | Fleet wants a captain to blame; longer watch shows advance and take, not a navy that declares war |
| *give us the longer watch* (ambiguous) | show us the years behind that claim / the watch product that made you sure |
| *yesterday's quiet proof of tomorrow's safety* | quiet is not the same as safe / missing returns Fleet cannot ignore |
| *Fleet prefers a single failure story* | point at the traffic wall: three clusters one way, one reverse — don’t force one clean cause |
| *the question Fleet procedure always asked when absences arrived without a recovery brief* | Look at the list. Fleet procedure was simple: if traffic stopped answering, ask whether anyone went looking. |
| *Low strategic value* (civilian / colony / grunt POV) | *routine* / *easy to skip on the schedule* / cut — Fleet staff labels belong in Fleet mouths or Fleet rooms |

**Rules:**

1. **Plain first.** If a line needs a second read to decode the metaphor, rewrite it in ordinary professional speech.
2. **Expand fog; do not decorate it.** One clear extra clause beats a stylish compression.
3. **Cut hyperbole and costume.** No saluting archives, enemy-with-a-face poetry, pride-as-tax, funeral rehearsals, weather they cannot command.
4. **Dialogue points at the work.** Prefer dates, clusters, gaps, search ships, strategic charts — not abstract “stories,” “frames,” or “narratives” unless a character is literally fighting a word.
5. **Narrator setups stay short and concrete.** *She looked at the list. Procedure was X. Then the question.* Not a thesis about what institutions always do.
6. **Keep blunt.** Vex/Senka/Calder may be hard; they should not be ornate.
7. **Match the mouth.** Colony grunts, relay clerks, and local civilians do not think in Fleet staff labels (*low strategic value*, *elevated threat*, *theater faction*). Say what they would actually notice.
8. **Test:** Could a tired watch officer understand the line on first hearing? If not, simplify and expand.
9. **Name the damn thing.** Do not rotate cute synonyms for the same board fact. If the Peligar advance reaches Segment approaches, say *the Peligar advance*. If systems are going quiet, say *systems going quiet*. Do not hide the referent behind *pressure*, *contour*, *the pattern*, *the edge*, *the line*, or other synonym bun fillers unless a character is literally fighting over what word to use on a status line.

**Search for regressions:** `narrative|salute|with a face|proof of tomorrow|failure story|recovery narrative|conceptual gap|payment completed|pride was expensive|without becoming a speech|That was as close|Liking is not required|Ugly phrases|war words|held the language line|Strike stewardship|smuggles|Not relief|Cost paid|experience, not poetry|Friendship was not on offer|Those were not the same sentence|did not invent significance`

**Hard cut (do not soften — delete):**

- Empty linguist banter that restates rails already locked (*war words*, *ugly phrases keep people honest*, *liking is not required*)
- Ellis / historian decorative word-strikes in early Act I when the dispute is not live (*shared losses*, *stewardship*, kinship policing)
- Caption endings after the beat already landed (*That was as close as…*, *Those were not the same sentence*, *It sounded like experience, not poetry*)
- Ritual-sounding custody shorthand (*spoke the chain aloud*) — say the real check in plain speech
- Clever two-noun traps (*trap shaped like caution* / *weapon shaped like science*) unless a character needs that exact hardness — prefer plain accusation names
- Triadic Council rhetoric (*Fleet Council will hear trap… theater…*) — name the concrete institutional ask or fear once, in plain speech
- Default fog nouns for the Peligar advance (*pressure*, *contour*, *the pattern*, *the line*) when the sentence means *Peligar advance*, *systems going quiet*, *missing returns*, or *advance marks*. Keep ordinary *under pressure* for stress. Keep *contour lines* only as literal chart drawing language, not as the name of the threat.
- Default *buffer* / *buffer posture* for the Fleet ask — prefer *treat warnings as actionable where Fleet ships already travel* / *Segment approaches* / *shared warning work*. Keep *buffer* only if a character is literally fighting that word on a draft.

### 3.3h Name the damn thing (no cute synonym rotation)

Distilled from Book Two Ch.3 author edit (2026-07-21).

**Rule:** When a sentence refers to a concrete board fact, **use the concrete name**. Do not invent a softer synonym and rotate it for variety.

| Fog / cute synonym | Name the thing |
|--------------------|----------------|
| *the pressure* / *contact pressure* | *the Peligar advance* / *we take the first hit* |
| *the contour* (as actor: reaches, advances, waits) | *the Peligar advance* / *systems going quiet* / *missing returns* |
| *the pattern* (when meaning the advance) | *the Peligar advance* or name what is on the board |
| *the line* / *the edge* (when meaning the advance) | *Peligar contact edge* / *advance marks* / *Peligar advance* |

**Allowed exceptions:**

- A character is **literally** fighting a status-line word (*processing* vs *aggression*, *cooperative warning posture*).
- Literal chart craft: *contour lines* drawn on a map product.
- Ordinary stress English: *under pressure*.
- Synonym buns (rare; earned; edible).

**Test:** If you swap the fog noun for the concrete name and the sentence gets clearer, keep the concrete name.

**Search for regressions:** `\bthe pressure\b|\bthe contour\b|moves the contour|contour keeps|contour will not|contour is real|inside the contour|expanding-contour|contact pressure|pressure maps|pressure marks`

### 3.3k Plain vocabulary: no odd synonyms, weird analogies, or substitute language

Distilled from Book Two Ch.3–4 author edits (2026-07-21 / 2026-07-22).

**Hard ban — unnecessary analogies and synonyms.** If a plain noun or verb already names the thing, do not decorate it with a clever comparison, a second synonym for variety, or a literary stand-in. Prefer the first true word. Cut the rest.

| Weak | Prefer |
|------|--------|
| *clean pipe for bad news* | the channel carried bad news / still no answer |
| *mourning with the volume down* | they were quiet / nobody spoke |
| *honesty were a person who had arrived late* | Cut. Keep the next line of dialogue. |
| rotating *silence* / *absence* / *emptiness* / *blank* for the same missing colony | pick one plain word and keep it |
| *hate a different clock* / *hours that already spent themselves* | hate yesterday / the time is already gone |

When the author has to ask *what does this mean?* the word choice has failed. This usually happens in one of three ways:

**1. Odd synonym rotation** — substituting a different word each time to avoid repetition, even when the first word was correct. **Do not do this for style.**

| Odd rotation | Keep the plain word |
|---|---|
| *notices* / *the posture* / *the ask* / *the product* (all meaning warnings) | *warnings* or name the concrete object — an unsafe-approach alert, a pressure map |
| *the instruments* cycling through *the posture*, *the agreement*, *the package* | name the specific thing signed or agreed |
| *the advance* / *the edge* / *the contour* / *the pressure* (all meaning the Peligar advance) | *the Peligar advance* — name it once, keep it |

**2. Weird, extended, or unnecessary analogies** — a comparison that needs explaining is not working. An analogy that is only stylish is also not working.

| Fails | Why | Fix |
|---|---|---|
| Analogy that takes longer to read than the point it makes | The analogy became the scene | Cut the analogy; state the point |
| Two different analogies for the same thing in one scene | Confusion, not clarity | Pick one or cut both |
| Analogy that invites the next character to ask what you mean | It failed | State the real thing |
| Analogy used only to sound literary (*honesty as a person*, *mourning with volume down*) | Style replaced sense | Cut; keep the action or dialogue |

**Rule:** If the analogy clarifies in one short clause, keep it. If it needs three sentences to land, it is not an analogy — it is a scene within a scene. Cut it or build it as its own beat. **If the sentence is clearer with no analogy at all, cut the analogy.**

**Brenner exception:** Brenner may use a literal maintenance story (engines, couplers, feeds) as the setup for a proposal. This works because the story *is the argument*, not a metaphor for the argument. Keep those. Do not extend them into recipe lists or flour-and-yeast asides.

**3. Substitute language** — a word standing in for a concrete thing because the writer did not want to commit to the real name.

| Substitute | Name the thing |
|---|---|
| *the ask* (policy level) | name what was asked: *a warning exchange*, *treaty authorization*, *Calder's proposal* |
| *the work* (vague) | name what the work is: *the lexicon*, *the survey order*, *the station design* |
| *the posture* | *how Fleet responds to Dominion warnings* / *the cooperative warning agreement* |
| *the framework* | name the actual document, instrument, or agreement |

**Search for regressions:** `\bthe ask\b|\bthe posture\b|\bthe framework\b|\bthe product\b|\bthe instrument\b` (when vague, not named) · `odd synonym|substitute|weird analogy`

**Test:** Read the line aloud. If you would pause and wonder what the word refers to, replace it with the concrete name.

### 3.4 Prefer concrete custody / work language

Institutions act through packets, custody, seals, status lines, classification stamps, and named procedures — not soft verbs that sound literary, and not cute shorthand pairs that only staff would love.

| Weak | Prefer |
|------|--------|
| Park holds the recording | Park has the recording in custody |
| Trust had not been purchased | Trust had not arrived / still absent |
| radioactive parts (if foggy) | contested / unresolved / disputed — or keep if the POV earned the metaphor |
| stop rules and shared nouns | walk-outs and word lists / who can suspend and what words may travel |
| who can walk / walk-out rights (shorthand) | who can end a session / who can leave the table / either side can still end the session |
| safe pedestal / watched from a pedestal (metaphor) | safe distance / from Helion / on the relay — keep *pedestal* only as the literal relay stand in the room |
| header / headers (default staff talk) | status line / message / classification / shared traffic — keep *header* only as a named field when specialists are literally editing one |
