# Character Generation

When creating or revising character profiles, load:

* `Writing_System/Style_Guide/global_writing_principles.md` — *Authentic Voice and Characterization*, *Romantic Interests*, *Language and Swearing*
* Existing sheets in `Characters/Primary_Crew/` and `Characters/Support_Crew/` as templates

Each sheet should include **AUTHENTIC VOICE & HUMANIZATION** (or **Authentic voice** for support crew) with blind-read test, identity influences, dialogue habits, romantic orientation, and language/oaths.
