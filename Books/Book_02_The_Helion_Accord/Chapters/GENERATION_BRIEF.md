# Book Two — Chapter Generation Brief

**Target:** 150,000–200,000 words total (Prelude + Ch. 1–27)  
**Theme:** Trust is harder than war  
**Time gap:** ~18 months after Book One  
**Outlines:** `Planning/Outline/`  
**Output:** `Chapters/prelude.md`, `Chapters/chapter_01.md` … `chapter_27.md`

## Priority (hard)

1. Character
2. World
3. Story
4. Tech (only when a character needs it)

## Voice

- Plain speak. Simple dialogue.
- Dumb down technical talk — professionals talk like people at work, not manuals.
- Same craft rules as Book One: `Writing_System/Style_Guide/global_writing_principles.md`, `solmare_craft_doctrine.md`, `chapter_final_edit_pass.md`, `chapter_editing_pass.md`.

## Ban / starve these

| Pattern | Rule |
|---------|------|
| Coffee/tea cold, warm, hot, too strong, gone cold | Never use drink temperature/fill as clock or mood |
| merge / merged / merging | Avoid unless unavoidable |
| file / filed / filing | Rare; prefer logged, noted, recorded, wrote down |
| whisper / whispered / whisper-class | Avoid |
| lane / lanes (overuse) | Prefer corridor, route, traffic, path — vary |
| Aphorisms, author winks, profound metaphors | Cut |
| Em dash spam | Prefer periods/commas |
| Colon dialogue tags (`Name: "…"`) | Never — use said/asked |
| Peligar POV interior | Forbidden |
| Peligar self-name / full motives / negotiation success | Withheld |

## Length

- Prelude: ~4,500–5,500
- Chapters: ~6,500–8,000 (let scene end the chapter; no padding)

## Continuity

- Read the chapter outline before writing.
- Read the previous chapter’s last ~40 lines for handoff.
- Update nothing in Book One.
- Dominion = tea; Fleet = coffee (presence OK; temperature clock forbidden).
- Spelling: **Peligar**
