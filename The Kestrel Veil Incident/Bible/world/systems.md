
# BOOK ONE STAR SYSTEMS

## 1. Helion Prime
Industrial core world, heavily militarized, fleet academy presence.

## 2. Varkon Drift
Resource extraction system; unstable asteroid belts interfere with sensors.

## 3. Sable Point
Quiet trade hub; rumored data laundering and signal interception activity.

## 4. Korrath Expanse
Deep survey system; officially mapped, but inconsistent readings exist.

## 5. Eryx Null Zone (RESTRICTED)
Navigation anomalies; ships report time distortion effects.

## 6. Unknown Designation (EDGE NODE)
Only partially charted. Mission origin point for Book One.
