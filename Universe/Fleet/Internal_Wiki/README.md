## Classification reminder

This wiki's **public pages** use Fleet's standing language for frontier findings: *provisional*, *unconfirmed*, *Neutral Zone corridor approach*, *anomaly*, *structured observation pending classification*.

Do **not** promote provisional Neutral Zone findings to confirmed civilizational status on public pages.

Where deeper truth requires post-authorization access, articles use:

> **[CLASSIFIED — post-authorization]**  
> Restricted text appears only in marked callouts.

See [Known Civilizations](Known_Civilizations.md) and [Frontier](Frontier.md).
